# código para Apikey 🔑

- crie um arquivo chamado config.json

  nele escreva:

{
    "apikey": "[sua chave da api]"
}

# como conseguir uma chave API 🔑❓


- entre em https://openweathermap.org/api
- crie uma conta (está na navbar)
- clique no seu nome 
- vá em "My API keys"

(lembre-se sempre de instalar as dependecias)
